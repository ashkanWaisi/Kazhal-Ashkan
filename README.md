# 💍 Ashkan & Kazhal — Luxury Wedding Welcome Board Website

> A premium, Awwwards-level wedding welcome board website crafted with pure HTML5, CSS3, and Vanilla JavaScript.

---

## ✨ Overview

This is a **luxury, production-ready wedding welcome board website** designed with the aesthetic polish of a five-star wedding invitation.  
It features elegant typography, smooth cinematic animations, a glassmorphism UI, countdown timer, interactive modals, and a golden ambient particle system.

---

## 📁 Project Structure

```
wedding-welcome/
│
├── index.html          ← Main page (all content)
├── style.css           ← Complete luxury stylesheet
├── script.js           ← All JavaScript functionality
├── README.md           ← This file
│
└── assets/
    └── hero.jpg        ← ⚠ Place the wedding photo here
```

---

## 🚀 Quick Start

1. **Extract** the ZIP archive
2. **Place** the wedding photo named `hero.jpg` inside the `assets/` folder
3. **Open** `index.html` in any modern browser
4. ✅ Done — no build tools, no npm, no dependencies!

## 🎵 Background Music (Optional)

The music button is fully wired. To enable:
1. Place a `.mp3` file at `assets/audio/wedding-music.mp3`
2. Uncomment the `<source>` tag in `index.html`:
   ```html
   <source src="assets/audio/wedding-music.mp3" type="audio/mpeg" />
   ```
